## [1.0.0] - Initial Release

### Added
- **100% Offline AI Assistant**: Ask questions and chat with your local documents using the on-device Gemma 4 model via MediaPipe.
- **Semantic Search Engine**: Fast and accurate document retrieval powered by ObjectBox Vector DB.
- **Background Indexing**: Automated scanning and indexing of local documents (PDFs, images, text notes) using WorkManager, optimized for battery life.
- **Search-First UI**: A minimalist, clean user interface built with Kotlin, Jetpack Compose, and Material 3 design principles.
- **Metadata Management**: Efficient storage and management of document metadata using Room (SQLite).

# Private Librarian - Sample Note

The Private Librarian application is an on-device AI assistant designed to index local PDFs, images, and notes. 

Key Features:
1. Privacy-first: All processing happens on-device.
2. Local LLM: Uses Gemma 4 for natural language answering.
3. RAG: Retrieval-Augmented Generation over your local files.
4. Offline Search: Semantic search without internet access.

Hardware Requirements:
- Minimum: 6 GB RAM
- Recommended: 8 GB+ RAM
- OS: Android 10+
